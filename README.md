## Stillwrk

![Stillwrk](screenshot.png)

A minimal focus timer with ambient video backgrounds. Built with React, TypeScript, and Vite.

## Features

- 25/5 Pomodoro timer that switches automatically
- Live clock
- 4 ambient video backgrounds - fireplace, ocean, rain, wind
- Clean glass-style UI

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS v4

## Getting Started

```bash
npm install
npm run dev
```

## Videos

Add your own MP4 files to the `public` folder:

```
public/
  fireplace.mp4
  ocean.mp4
  rain.mp4
  wind.mp4
```
